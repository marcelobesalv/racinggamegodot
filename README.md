- **Engine:** Godot 3.5.2
- **Type:** 3D Racing Game
- **Year:** 2023

## Description

A 3D racing game prototype built with Godot Engine. The project includes:
- Car physics and controls
- Track system with curved and straight sections
- Racing environment with barriers
- Menu system

## Running the Project

1. Download and install [Godot 3.5.2](https://godotengine.org/download/3.x/)
2. Clone this repository
3. Open the project in Godot 3.5.2
4. Press F5 to run the game

## Project Structure

- `/Scenes/` - Track and terrain scenes
- `/Asssets/` - Models, textures, and materials
- `Menu.tscn` - Main menu scene
- `Scene1.tscn` - Main game scene
